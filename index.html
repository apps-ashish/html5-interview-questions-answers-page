Top 50 HTML5 Interview Questions & Answers

1. What is HTML5?

HTML5 is the latest version of the Hypertext Markup Language used to structure web content and support multimedia without plugins.

2. What are new features in HTML5?

New semantic tags, audio/video support, canvas, local storage, geolocation, etc.

3. What are semantic elements in HTML5?

Elements that convey meaning like <header>, <footer>, <section>, <article>, etc.

4. What replaced <frame> and <frameset> in HTML5?

These are obsolete. Use <iframe> or modern layout with CSS.

5. What is the purpose of the <canvas> element?

For drawing graphics on the fly via JavaScript.
